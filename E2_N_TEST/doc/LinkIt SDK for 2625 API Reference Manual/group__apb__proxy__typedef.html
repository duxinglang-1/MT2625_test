typedef apb_proxy_status_t(* apb_proxy_at_cmd_handler) (apb_proxy_parse_cmd_param_t *p_parse_cmd)

The APP registers a callback function for a specific AT command.

Don't do any CPU heavy calculation. Don't call timer delay function.

Parameters
[in]p_parse_cmdA pointer to AT command data. APB proxy is responsible for managing the memory which the pointer points to.
Returns
None.

typedef void(* apb_proxy_data_mode_event_callback_t) (apb_proxy_event_type_t event, apb_proxy_data_mode_event_t *pdata)

The APP registers a callback function to the APB using apb_proxy_create_data_mode() to transfer user data or binary data.

Don't do any CPU heavy calculation.

Parameters
[in]eventThe triggerred event type for data mode.
[in]pdataA pointer to the event data. The APB proxy is responsible for managing the memory which the pointer points to.
Returns
None.
