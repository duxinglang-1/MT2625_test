enum apb_proxy_at_cmd_result_code_t

This enumeration defines the AT Command result code.

The output string for the specific result code is also provided. For example, when the result code is APB_PROXY_RESULT_NO_CARRIER, the output string is "NO CARRIER". Please note, there is no output string for APB_PROXY_RESULT_PROCEEDING and APB_PROXY_RESULT_UNSOLICITED.

Enumerator
APB_PROXY_RESULT_OK 

OK (The operation completed successfully.)

APB_PROXY_RESULT_CONNECT 

CONNECT (Data connection setup completed successfully.)

APB_PROXY_RESULT_RING 

RING (Ring indication.

Not supported in NBIOT.)

APB_PROXY_RESULT_NO_CARRIER 

NO CARRIER (Data connection ended or setup has failed.)

APB_PROXY_RESULT_ERROR 

ERROR (An error occurred.)

APB_PROXY_RESULT_NO_DIALTONE 

NO DIALTONE (Voice call network has failed.)

APB_PROXY_RESULT_BUSY 

BUSY (The system is busy with another command.)

APB_PROXY_RESULT_NO_ANSWER 

NO ANSWER (Voice call user not answering.)

APB_PROXY_RESULT_PROCEEDING 

No corresponding output string (Intermediate response.)

APB_PROXY_RESULT_UNSOLICITED 

No corresponding output string (Unsolicited response.)

APB_PROXY_RESULT_CUSTOM_CONNECT 

The user fills hte customed response output when data connection sets up.

APB_PROXY_RESULT_CUSTOM_ERROR 

The user must fill the customed error output.

This is final response code.

enum apb_proxy_cmd_mode_t

This enumeration defines the AT command operation mode.

Enumerator
APB_PROXY_CMD_MODE_ACTIVE 

Active mode command, such as "AT+CMD".

APB_PROXY_CMD_MODE_EXECUTION 

Execute mode command, such as "AT+CMD=<op>".

APB_PROXY_CMD_MODE_READ 

Read mode command, such as "AT+CMD?".

APB_PROXY_CMD_MODE_TESTING 

Test mode command, such as "AT+CMD=?".

APB_PROXY_CMD_MODE_INVALID 

The input command doesn't belong to any of the four types.

enum apb_proxy_event_type_t

This enumeration defines APB event type for data mode.

Enumerator
APB_PROXY_USER_DATA_IND 

New user data is ready.

APB_PROXY_STOP_SEND_USER_DATA 

Stop sending user data for data flow control.

APB_PROXY_RESUME_SEND_USER_DATA 

Resume sending user data for data flow control.

APB_PROXY_DATA_MODE_CLOSED_IND 

Low layer has closed the data mode.

APB_PROXY_DATA_MODE_TEMP_DEACTIVEED_IND 

Low layer has closed the data mode temporarily.

APB_PROXY_DATA_MODE_RESUMED_IND 

Low layer has resumed to data mode.
