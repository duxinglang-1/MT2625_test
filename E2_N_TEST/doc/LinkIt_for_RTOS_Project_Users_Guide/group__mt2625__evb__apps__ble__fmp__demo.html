Overview

Overview
  • Application description
    • This application demonstrates BLE Find Me Profile features based on MediaTek MT2625 SOC for RTOS development platform. The architecture diagram is shown below.
    • Application features
      • Find Me Profile (FMP). Allows users to find the misplaced device using Bluetooth LE.
      • The application features are independent, if not otherwise mentioned.
Hardware and software environment
  • Supported HDK
    • 2625 FPGA.
  • HDK switches and pin configuration
    • N/A.
  • Environment configuration
    • Please refer to FPGA & IVP load user guide
  • Customization options and usage
    • No compile option is available to disable an application feature.
    • To disable a feature, delete the feature initialization function in the main.c.
Directory contents
  • Source and header files
    • inc: Common header files.
    • src/ble_find_me_server: Find Me server source code.
    • src/main.c: Main program.
    • src/ept_gpio_var.c: The GPIO configuration file generated by the Easy Pinmux Tool (EPT).
    • src/ept_eint_var.c: The EINT configuration file generated by the EPT.
    • src/system_mt2625.c: The configuration file of the Cortex-M4 with floating unit MCU core registers and system clock.
    • src/bt_app_common.c: Bluetooth application common file.
    • src/bt_gattc_discovery.c: GATT service discovery file.
    • src/task_def.c: Task definition file.
    • src/bt_init.c: Bluetooth initialization file.
    • src/gatt_service.c: GATT and GAP service file.
    • src/hci_log.c: Display HCI log.
  • Project configuration files using GCC
    • GCC/feature.mk: Feature configuration file.
    • Makefile: Makefile.
    • GCC/flash.ld: Linker script.
    • GCC/syscalls.c: MT2625 syscalls for GCC.
    • GCC/startup_mt2625.s: MT2625 startup file for GCC.
Run the application
  • How to build the ble_fmp_demo application
    • GCC version
      • make command "./build.sh mt2625_evb ble_fmp_demo bl" under the SDK root folder.
  • How to download the headset_ref_design application
    • Install and use MT2625 Flash Tool to download the application binary ble_fmp_demo.bin.
  • How to run each feature
    • Power on the MT2625 EVB to initialize and run the features in the background.
    • FMP server and Client can also be triggered on the iphone. Find more details in each sub-feature project's readme file.
