ble_find_me_server
mt2625_evb » apps » ble_fmp_demo
Overview
  • Feature description
    • This feature demonstrates how to find a local device through the Find Me Profile using Bluetooth LE.
Hardware and software environment
  • Please refer to mt2625_evb/apps/ble_fmp_demo/readme.txt.
Directory contents
  • Source and header files
    • inc: Common header files.
    • inc/ble_find_me_server: Find Me server feature header files.
    • src/bt_app_common.c: Bluetooth support source code.
    • src/ble_find_me_server/ble_find_me_server.c: Find Me server feature source code.
Run the feature
  • Build and Download
    • Please refer to mt2625_evb/apps/ble_fmp_demo/readme.txt.
  • Install the application "LightBlue" from the App Store on iphones and launch the application, then select "Find Me" in the Virtual peripherals List.
  • Power on the MT2625 EVB, then BLE will enable Advertising.
  • Click the MT2625 EVB named like "ble_fmp_demo" in the peripherals nearby list on the "LightBlue" Application, then iphone will connect to the MT2625 EVB.
  • After the Bluetooth LE connection is established, write a new alert level like "0x02" or "0x01" on the "LightBlue" application to start an alert.
  • The MT2625 EVB displays a log "ble_fmp_enable_alert: enable alert with level is %d success!".
  • Then, write a new alert level like "0x00" on the "LightBlue" application to stop an alert.
  • The MT2625 EVB will display "ble_fmp_disable_alert: disable alert success!".
